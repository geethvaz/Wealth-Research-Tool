PROJECT: Script Research — Internal Wealth Management Research Tool

TECH STACK (mandatory, do not deviate):
- Next.js 14 with App Router
- Tailwind CSS
- shadcn/ui components
- Recharts for all charts
- TypeScript throughout
- Standard npm packages only — no Replit-native integrations

DO NOT USE under any circumstances:
- ReplDB or any Replit key-value store
- Replit Object Storage
- Replit Auth
- Replit Deployments configuration
All code must be fully portable and deployable on Vercel 
without modification.

ENVIRONMENT VARIABLES (reference only, never hardcode):
- DATABASE_URL (Neon PostgreSQL)
- ANTHROPIC_API_KEY
- ONEDRIVE_CLIENT_ID
- ONEDRIVE_CLIENT_SECRET
- ONEDRIVE_TENANT_ID

DATABASE: All persistence uses Neon (PostgreSQL) via the 
'postgres' npm package connected through DATABASE_URL.

FILE STORAGE: No permanent file storage on server. 
Excel files are either downloaded directly by the user 
or pushed to OneDrive via Microsoft Graph API.

DEPLOYMENT TARGET: Vercel. Use standard Next.js 14 
App Router folder structure. No custom build configs.
